## Passionate Navigation

#### Getting Started
Requires:
- Ruby 2.6.3

To setup run:
- `bundle install`
- `rails db:create && rails db:migrate`

The documentation is generated using:
- `rake rswag`

#### Details
The API responses correspond to the jsonapi specification that can be found here: https://jsonapi.org/

The api documentation is generated using swagger from requests specs and can be found on the root path of the app: `/`
